# Data Structures Course - Lab Solutions

This repository contains 11 lab assignments covering fundamental data structures and algorithms. Each lab includes Java implementations with test cases.

## Labs Overview

### Lab 01 - OOP Inheritance
Object-oriented programming concepts with inheritance hierarchies.
- **Files**: Book.java, Student.java, Graduate.java, Undergraduate.java, Textbook.java
- **Concepts**: Classes, inheritance, polymorphism, abstraction

### Lab 02 - Singly Linked Lists
Implementation and operations on singly linked lists.
- **Files**: SLL.java, SLL_Driver.java, LinkedListDriver.java
- **Concepts**: Node creation, traversal, insertion, deletion, searching

### Lab 03 - Doubly Linked Lists
Implementation and operations on doubly linked lists.
- **Files**: DLL.java, DLLNode.java, DLLTest.java
- **Concepts**: Bidirectional traversal, forward and backward pointers

### Lab 04 - Stacks and Queues
Implementation of stack and queue data structures with applications.
- **Files**: LabStack.java, LabQueue.java, BalancedParentheses.java, Postfix.java, ReverseQ.java
- **Concepts**: LIFO/FIFO operations, balanced parentheses checking, postfix evaluation

### Lab 05 - Array-Based Queues
Queue implementation using arrays.
- **Files**: QueueAsArray.java, QueueAsArrayTest.java, SLL.java
- **Concepts**: Circular queue, enqueue, dequeue operations

### Lab 06 - Binary Search Trees
Binary search tree implementation and traversal methods.
- **Files**: BST.java, BSTNode.java, BinaryTreeTraversal.java, Stack.java, Queue.java
- **Concepts**: In-order, pre-order, post-order traversal, search, insertion

### Lab 07 - AVL Trees
Self-balancing binary search trees.
- **Files**: AVLTree.java, TestAVL.java
- **Concepts**: Tree balancing, rotations, maintaining height balance

### Lab 08 - Binary Heaps
Min/max heap implementation with heap operations.
- **Files**: BinaryHeap.java, Hospital.java
- **Concepts**: Heap property, insertion, deletion, heapify operations

### Lab 09 - Hash Tables
Hash table implementation with collision handling.
- **Files**: HashTable.java, Entry.java
- **Concepts**: Hashing, collision resolution, load factor

### Lab 10 - String Matching (KMP)
Knuth-Morris-Pratt algorithm for pattern matching.
- **Files**: KMPImplementation.java, NextArray.java, PSuffix.java, Task1.java, Task2.java
- **Concepts**: Pattern matching, failure function, string algorithms

### Lab 11 - Graphs
Graph representations and traversal algorithms.
- **Files**: Graph.java, GraphDriver.java, Edge.java, GraphLAB.java
- **Concepts**: Adjacency list/matrix, BFS, DFS, graph traversal

## Getting Started

1. Navigate to the desired lab folder
2. Compile and run the Java files with your preferred IDE or command line
3. Review the test files to understand the expected behavior

### Compilation Example
```bash
javac *.java
java ClassName
```
